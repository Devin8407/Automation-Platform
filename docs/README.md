# Documentation

This directory contains the project's technical documentation.

## Structure

### Architecture

System design, architectural principles, and high-level design documentation.

- [Architecture Overview](architecture/overview.md)

### Architecture Decision Records (ADRs)

Architectural decisions and the reasoning behind them.

- [ADR Index](adr/README.md)

## Documentation Philosophy

Documentation is written alongside development and evolves with the project.

The goal is to document not only **how** the system works, but also **why** important architectural decisions were made.
